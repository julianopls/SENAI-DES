## 🚀 Como Usar

```bash
# Clone o repositório
git clone https://github.com/julianopls/SENAI-DES.git

# Acesse a pasta
cd SENAI-DES

# Navegue pelo semestre desejado
cd "3º Semestre"
```

Para projetos HTML, basta abrir o arquivo `.html` no seu navegador.  
Para projetos C++, compile com `g++` ou use o CMake disponível.
